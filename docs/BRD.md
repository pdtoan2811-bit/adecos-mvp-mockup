# Business Requirement Document (BRD) - Adecos MVP

## 1. Executive Summary
Adecos is an AI-powered research assistant designed to help affiliate marketers discover and analyze high-potential affiliate programs within specific niches. The MVP focuses on delivering a premium, streamlined experience ("No Spec, No Service").

## 2. Problem Statement
Affiliate marketers spend hours manually searching Google, reading listicles, visiting brand pages to find commission rates, and verifying legitimacy. Data is unstructured and hard to compare.

## 3. Solution
An AI application that:
- Takes a niche as input.
- Aggregates top affiliate programs.
- Extracts key metrics (Commission, Traffic, Terms, Legitimacy).
- Presents data in a structured, sortable table.

## 4. MVP Scope
- **Simulation**: Instead of live crawling, use Gemini API to generate realistic mock data for immediate value demonstration.
- **Region**: Vietnamese language interface and focus.
- **Design**: Premium "Glassmorphism" aesthetic with serif typography.
- **Platform**: Web Application (React + FastAPI).

## 5. Success Metrics
- streaming latency < 2s.
- "Wow" factor in UI design (Glassmorphism).
